plang annotate — Add notes, sources, assumptions, or risk annotations to a spec item.
plang split — Break a large document into smaller tagged units or per-object files.

plang index — Build or refresh a local search index for fast document lookup.
plang stats — Report counts, coverage, defect density, fuzzy terms, and other repository metrics.
plang graph — Visualize links between tags, sources, owners, assumptions, and dependent specs.
plang doctor — Diagnose workspace, config, parser, or indexing problems and suggest fixes.
plang config — View or manage CLI configuration, defaults, and workspace settings.
plang completion — Generate shell completion scripts for supported shells.
plang export — Render Planguage content into JSON, CSV, HTML, or other downstream formats.

plang help — Show help for commands, flags, concepts, and common task workflows.
plang version — Print the CLI version, build info, and supported capability set.
